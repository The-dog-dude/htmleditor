# HTML Editor
## HTML Editor
HTML Editor is a simple HTML editor.
## Features
* Live preview
* Syntax highlighting
* Auto Tag Closing
* Auto Syntax Completion

```
git clone https://github.com/Varshithvhegde/htmleditor.git
```
## Usage
```javascript
var editor = new HTMLEditor();
editor.render();
```
